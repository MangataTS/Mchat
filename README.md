# Mchat
## 关于本次项目
本次项目是基于安卓写的一个APP，能够实现在指定ip和端口下进行即时通讯，由于时间比较紧，我还有很多功能没有写出来，有时间会继续写新东西的，本次项目的服务器端的和客户端都是我一个人完成的，所以可能会有点简陋，请见谅，下次会写更好的^^
&nbsp;
## 服务器的构建
服务器是我自己写的一个解析Socket数据的一个东西，但是服务器端也有几个指令，目前只实现了四个功能:
&nbsp;
1.服务器广播 @all

&nbsp;
2.关闭服务器 shutdown

&nbsp;
3.踢出指定端口的用户 kick

&nbsp;
4.清理不在线的用户(释放资源) cls
## 安卓端的构建
安卓端只有两个Activity，但是是以第二个Chatroom为主，进行聊天活动的，前面的MainActivity是更改用户名和ip地址还有端口的，你修改成你的ip和port就行
